### MC_Entity

### Layer Changes

Layer Changes allow you to change what layer a block renders on.

#### Representation

```java
layer.solid=<blocks>
layer.cutout=<blocks>
layer.cutout_mipped=<blocks>
layer.translucent=<blocks>
```

#### Implementation Support

* ❌ ShadersMod
* ✔️ OptiFine
* ✔️ Iris

Note: Optifine Only Has Partial Support
